# Abajo e' la Línea

Sitio web de Abajo e' la Línea, TV comunitaria de Temuco.

## Estructura

- `index.html` — marcado
- `style.css` — estilos
- `script.js` — reproductor en vivo (HLS)
- `logo.png` — logo del canal

## Uso

Abrir `index.html` en el navegador. El streaming en vivo requiere conexión a internet.
